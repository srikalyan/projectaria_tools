Aria Client SDK API Reference

Overview​

This page provides a list of Project Aria Client SDK's APIs.

Global Functions & Methods​

FunctionTypeDescription
set_log_level(level: Level)NoneSets the SDK logging verbosity

Classes​

aria.sdk.Error​

PropertyTypeDescription
codeintError code
messagestrHuman readable error message

aria.sdk.RecordingConfig​

PropertyTypeDescription
profile_namestrSensors profile name for recording

aria.sdk.StreamingSecurityOptions​

PropertyTypeDescription
use_ephemeral_certsboolUse ephemeral certs instead of persistent ones
local_certs_root_pathstrLocal directory path where streaming certificates are stored

aria.sdk.StreamingConfig​

PropertyTypeDescription
security_optionsStreamingSecurityOptionsSecurity options used to start streaming
topic_prefixstrUse this as a unique string to prefix all streamed data messages
profile_namestrSensors profile name used to start streaming
streaming_interfaceStreamingInterfaceNetwork interface used to start streaming

aria.sdk.StreamingSubscriptionConfig​

PropertyTypeDescription
security_optionsStreamingSecurityOptionsSecurity options used to subscribe to an existing live secure stream
subscriber_data_typeStreamingDataTypeData types to subscribe to
message_queue_sizeintSize for the message queue. A shorter queue size may be useful if the processing callback is always slow and you wish to process more recent data
subscriber_namestrRetrieve the subscriber name
subscriber_topic_prefixstrRetrieve the topic used to prefix the existing live stream

aria.sdk.DeviceInfo​

PropertyTypeDescription
boardstrDevice board name
bootloaderstrBootloader version
brandstrDevice brand name
manufacturerstrManufacturer name
modelstrModel name
productstrProduct name
serialstrSerial number
timeintOS build timestamp

aria.sdk.DeviceStatus​

PropertyTypeDescription
battery_levelintBattery level
charger_connectedboolUSB charger cable state
chargingboolCharging state
wifi_enabledboolWiFi activation state
wifi_configuredboolWiFi configuration state
wifi_connectedboolWiFi connection state
wifi_ip_addressstrWiFi IP address
wifi_device_namestrWiFi device name
wifi_ssidstrWiFi SSID name
logged_inboolCompanion App user login state
developer_modeboolDeveloper mode state
adb_enabledboolADB state
thermal_mitigation_triggeredboolIndicate max level temperature has been reached triggering throttling
skin_temp_celsiusfloatDevice temperature
default_recording_profilestrDefault recording profile used when pressing the top right HW button
is_recording_allowedboolRecording capability state
device_modeDeviceModeDevice mode

aria.sdk.DeviceClientConfig​

PropertyTypeDescription
ip_v4_addressstrIP v4 address to use for connecting to the device via Wi-Fi
device_serialstrDevice serial number used when connecting to the device via USB (only necessary if multiple devices are plugged in)
adb_pathstrSpecify your own custom ADB version
reconnection_attemptsintNumber of reconnection attempts before time out. Defaults to 2

aria.sdk.DeviceClient​

MethodTypeDescription
create(config: DeviceClientConfig)DeviceClientCreate DeviceClient instance
authenticate()NoneAuthenticate your client using the specified config
connect()DeviceConnect to device via Wifi or via USB using the specified config. Setting both ip address and adb path will lead to the adb path being ignored
disconnect(device: Device)NoneDisconnect Device instance

aria.sdk.Device​

MethodTypeDescription
recording_manager()RecordingManagerAccess recording capabilities
streaming_manager()StreamingManagerAccess streaming capabilities
info()DeviceInfoRetrieve device information such as device name and serial number
status()DeviceStatusRetrieve device status such as battery level and device temperature
factory_calibration_json()strRetrieve device factory calibration as JSON string

aria.sdk.RecordingManager​

MethodTypeDescription
start_recording()NoneStart recording
stop_recording()NoneStop recording
sensors_calibration()strRetrieve the device calibration computed from the sensors profile used to record
PropertyTypeDescription
recording_configRecordingConfigUsed to configure recording parameters such as the sensors profile
recording_stateRecordingStateDetermine current recording state
recording_profilesList[str]Returns a list of existing profile names to use to start recording

aria.sdk.StreamingManager​

MethodTypeDescription
start_streaming()NoneStart streaming
stop_streaming()NoneStop streaming
sensors_calibration()strRetrieve the device calibration computed from the sensors profile used to stream
PropertyTypeDescription
streaming_configStreamingConfigUsed to configure streaming parameters related to network interface, security, and sensors profile
streaming_stateStreamingStateDetermine current streaming state
streaming_profilesList[str]Returns a list of existing profile names to use to start streaming

aria.sdk.BaseStreamingClientObserver​

MethodTypeDescription
on_streaming_client_failure(reason: ErrorCode, message: str)NoneRetrieve streaming failure
on_image_received(image_and_record: projectaria_tools.core.sensor_data.ImageDataAndRecord)NoneRetrieve image data streamed from rgb, slam1, slam2 or eye tracking camera sensors
on_audio_received(audio_and_record: projectaria_tools.core.sensor_data.AudioDataAndRecord)NoneRetrieve audio data streamed from microphone sensors
on_imu_received(motion_data: List[projectaria_tools.core.sensor_data.MotionData], imu_idx: int)NoneRetrieve imu data streamed from IMU1 and IMU2 sensors. Use imu_idx to determine the IMU
on_magneto_received(magneto_data: projectaria_tools.core.sensor_data.MotionData)NoneRetrieve magnetometer data streamed from magnetometer sensor
on_baro_received(baro_data: projectaria_tools.core.sensor_data.BarometerData)NoneRetrieve barometer data streamed from barometer sensor

aria.sdk.StreamingClient​

MethodTypeDescription
create()StreamingClientCreate StreamingClient instance
subscribe()NoneSubscribe to data streamed from Aria
unsubscribe()NoneUnsubscribe to data streamed from Aria
set_streaming_client_observer(observer: StreamingClientObserver)NoneSets the observer to subscribe to the data streamed from Aria
is_subscribed()boolReturns streaming subscription state
